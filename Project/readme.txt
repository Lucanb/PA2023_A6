Tema Proiect Java - Nastasa Baras Luca A6

Realizarea unei aplicatii bancare ce are ca scop stocarea si gestionarea mai multor creditori.
Astfel am avea urmatoarele specificatii:
  -autentificare : admin/angajat + interfata grafica user-friendly
  -baza de date complexa care se ocupa cu gestiunea diferitor formularea (model relational)
  -salvare de fisiere csv + introducerea lor in baza de date
  -generare de contracte in functie de datele introduse
  -conditii de siguranta(verificarea datelor) + stocare in siguranta a datelor
  -Realizarea unui API-REST pentru informatii clienti/request-uri  
  -Infrastructura va functiona pe o retea(fiecare angajat este un thread in server)
  -folosire Java Spring
Bonus (posibil sa mai fac):
  -posibil sa fac optionea de related (algoritm pe grafuri) care sa imi afiseze diferite persoane related si sa usureze munca angajatului
  -pentru algoritmul de ralated posibil sa folosim si paralelizare
  -serverul sa aiba o infrastructura de microservicii gestionata de un Api-Gateway
  -Unit Testing(JUnit)
  -Statistica code covrege(Sonar Cube)

La final : Va fi intocmit un raport si o analiza a eficientei aplicatiei + posibil unit testing pe niste seturi de date
